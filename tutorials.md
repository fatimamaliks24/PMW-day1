## 2. Photogrammetry and 3D Reconstruction Explained – Computerphile
**Link:** https://www.youtube.com/watch?v=YE9rEQAGpLw

### Concepts Learned
- A 3D model can be created from multiple photographs taken from different angles.
- Structure from Motion (SfM) estimates camera positions and reconstructs scene geometry.
- Point clouds represent objects and environments using thousands of 3D points.
- Multi-view stereo techniques can create denser and more detailed 3D models.

### How I Applied It
I researched and compared several 3D reconstruction methods:
- COLMAP / Photogrammetry
- NeRF (Neural Radiance Fields)
- Gaussian Splatting
- Monocular Depth Estimation
- Multi-View Stereo

I also ran small experiments and generated output images to better understand these techniques.
